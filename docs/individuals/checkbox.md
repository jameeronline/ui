# Checkbox
<mark>Last Updated on: {docsify-updated}</mark>

> Checkbox group have been categorized as follows:
> 
> * Inline Group - use **list-inline** class in `<ul>`
> * Block Group - use **list-unstyled** class in `<ul>`
